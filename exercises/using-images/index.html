Using Images

The <img> tag

The <img> tag is the simplest and most common way to display an image in an html page. üòä It requires a src attribute, which indicates where the browser will find the image, and an alt attribute, which provides a textual version of the image for screen-readers.

Example: 

<img src="path/to/image.jpg" alt="My image">

Sizing an <img>

<img> tags can be sized with CSS. Generally, it is best to set only width or height, not both.

Cropping an <img>

We can crop by forcing width/height and using object-fit: cover;
